# commgame_eyetracker
Code for preprocessing and analyzing eye tracker data collected in the Communication Game experiments

UNDER DEVELOPMENT, NON-FUNCTIONAL YET

## Background:
In the Communication Game experiments, pairs of participants engage in various communication tasks while they are equipped with Pupil Core wearable eye trackers. Further recordings are also collected (EEG, mocap, audiovisual). Pair members are seated in separate labs. Recording PCs are synched to a common time via NTP over local network.
